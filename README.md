# InsightAI - Social Sentiment & Trend Predictor

## Project Overview
This project predicts the sentiment of social media posts using Machine Learning.

The model classifies tweets into:
- Positive
- Negative
- Neutral
- Irrelevant

## Dataset
- twitter_training.csv
- twitter_validation.csv

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Joblib

## Machine Learning Model
- TF-IDF Vectorizer
- Logistic Regression

## Model Accuracy
79.27%

## Features
- Dataset Loading
- Exploratory Data Analysis (EDA)
- Data Cleaning
- Sentiment Prediction
- Model Saving
- Data Visualization
