# InsightAI - Social Sentiment & Trend Predictor

## 📌 Project Overview
This project is a Machine Learning-based sentiment analysis system that predicts the sentiment of social media posts.

The model classifies tweets into:
- Positive
- Negative
- Neutral
- Irrelevant

---

## 📊 Dataset
- twitter_training.csv
- twitter_validation.csv

---

## 🛠 Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Joblib

---

## 🤖 Machine Learning Model
- TF-IDF Vectorizer (Text Feature Extraction)
- Logistic Regression (Classification Algorithm)

---

## 📈 Model Accuracy
- **79.27%**
